# Press Key

## Parameters

- **[Select Key]**: Corresponds to the virtual keys at the bottom of the Android phone.
    - Home
    - Back
    - Switch App

## Output

- **Output Parameters**
    - None
- **Execution Log**
    - Start Time
    - End Time
    - Duration (ms)
    - Status (Success/Failure)
