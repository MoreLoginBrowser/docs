### Zapier Integration

## Step 1: Search for "PhoneGrid" in the Zap Node

## Step 2: Configure the Zapier Node

### 1. Setup Configuration
**Action Event**
- Run Market Template One-Time Task
- Run Custom Personal Template One-Time Task
- Schedule Recuring Market Task

**Account**

You need to configure this the first time. After that, your account information will be saved and can be used directly in the future.
Clicking it will pop up a dialog asking you to enter "API ID", "API KEY", and "Grant Type": "client_credentials" (default).

[How to get API ID and API KEY?](https://guide.phonegrid.com/api-reference/open-api#get-started-with-phonegrid-open-apis)

### 2. Configuration
Below is an example using "Run Market Template One-Time Task" with configuration instructions:

**Cloud Profile**: Select the cloud phone to run

**Schedule Name**: Enter the task name

**Template Id**: Enter the template ID. How to find the template ID?

**Template Parameter**: Enter the template parameters in JSON format. How to view template parameters?

**Description**: Enter the task description (optional)
