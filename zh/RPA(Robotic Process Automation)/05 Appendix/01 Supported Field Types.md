# Supported Field Types

### RPA currently supports the following field types:

| Field Type    | Icon                                                                                                               | Description                       | Example                       |
| ------------- | ------------------------------------------------------------------------------------------------------------------ | --------------------------------- | ----------------------------- |
| String        | <img src="../image/filed/string.png" style="width:20px; height:20px; vertical-align: middle; margin:0 2px;" />     | String                            | "hello"                       |
| Integer       | <img src="../image/filed/int.png" style="width:20px; height:20px; vertical-align: middle; margin:0 2px;" />        | Number, supports decimals         | 123.45                        |
| Number        | <img src="../image/filed/num.png" style="width:20px; height:20px; vertical-align: middle; margin:0 2px;" />        | Integer                           | 123                           |
| Boolean       | <img src="../image/filed/bool.png" style="width:20px; height:20px; vertical-align: middle; margin:0 2px;" />       | Boolean                           | true/false                    |
| JSON          | <img src="../image/filed/obj0.png" style="width:20px; height:20px; vertical-align: middle; margin:0 2px;" />       | JSON                              | {"name": "hello", "age": 123} |
| ElementObject | <img src="../image/filed/element.png" style="width:20px; height:20px; vertical-align: middle; margin:0 2px;" />    | Clickable element, parsed as JSON |                               |
| Array         | <img src="../image/filed/icon_array.png" style="width:20px; height:20px; vertical-align: middle; margin:0 2px;" /> | Array                             | ["hello", "world"]            |
