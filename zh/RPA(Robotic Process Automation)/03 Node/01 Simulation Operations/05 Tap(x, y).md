# 点击（x，y）

RPA提供了3种模拟点击的方式：

1. [Tap Element](../01%20Simulation%20Operations/04%20Tap%20Element.md)：通过UI树的DOM元素找到并单击当前元素。
2. [Tap(x, y)](../01%20Simulation%20Operations/05%20Tap(x,%20y).md)：通过当前屏幕的X、Y轴坐标定位点击位置。
3. [通过文字识别（OCR）点击](../01%20Simulation%20Operations/06%20Click%20via%20OCR.md)：通过识别当前屏幕上的文字位置来点击。


下面详细介绍 Tap(x, y) 函数。

## 参数

- **[坐标]**
    - X（数字）
    - Y（数字）

    您可以通过【开发者工具】(../../02%20Using%20RPA/04%20Developer%20Tools.md)获取云手机中的X、Y坐标值。

- **[坐标随机偏移]**（数字）

    默认为0。设置偏移值后，每次点击位置都会在偏移范围内随机上、下、左、右偏移。

    例如，设置X：50，Y：100，随机坐标偏移量为10，则X的范围为40px-60px，Y的范围为90px-110px。

- **[点击类型]**
    - 单击
    - 双击
    - 长按
        - 选择长按时，您可以设置**[按下持续时间]**（以毫秒为单位）。

- **[点击后延迟]**

## 输出

- **输出参数**
    - 无

- **执行日志**
    - 开始时间
    - 结束时间
    - 持续时间（毫秒）
    - 状态（成功/失败）

## 示例

例如，在云手机桌面上，有一个Google Play商店。通过开发者工具查到中心点坐标为（X：927，Y：1326）。参数如下：

|关键|说明|
| :---------------------------- | :------------------ |
| **坐标** | X：“927”，Y：“1326”|
| **坐标随机偏移** | 0 |
| **点击类型** |单击 |
| **点击后延迟** | 0 |
