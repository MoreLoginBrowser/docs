# 输入文本

## 参数

- **[输入选择器类型]** 下拉菜单
    - 通过查找元素
    - 使用上一个元素

- 您可以在**[满足条件时]**中添加所需条件。支持以下4种条件，最多同时支持4个并行过滤器：
    - 文字
    - 完整ID
    - 类
    - 描述

    您可以通过开发者工具获取当前云手机屏幕的元素属性。

- **[输入字段超时]**（数字）
    - 填写数字后，当前查找器将轮询您需要查找的元素。找到元素后继续执行。如果在这段时间内没有找到该元素，则开始执行下一个节点。
    - 可以填写或引用一个数字，单位是毫秒。

- **[输入文本]**（字符串）
    - 输入文本字符数限制为 2000 个字符。

- **[键入前清除字段]** 开关
    - 开：输入文本前清除输入框的内容（默认）。
    - 关闭：如果输入框中有内容，则将当前输入的文本追加到原来的内容中。

- **[输入后按Enter]**切换
    - 开：输入文本后单击 Enter。
    - 关：输入文本后结束（默认）。

- **[输入后延迟]**（数字）

## 输出

- **输出参数**
    - Element：作为定位节点输出。您可以在后续过程中引用该节点或解析该节点内容（JSON）。可以通过输入的节点信息来判断定位是否正确。


- **执行日志**
    - 开始时间
    - 结束时间
    - 持续时间（毫秒）
    - 状态（成功/失败）

## 示例

例如，在云手机桌面上，希望在Google搜索框中输入“morelogin反检测浏览器”。通过开发者工具找到搜索框的fullID，如“com.android.quicksearchbox:id/search_src_text”。

配置如下：

|关键|说明|
| :------------------------------ | :-------------------------------------------------------------------------- |
| **选择类型** |通过查找元素 |
| **当条件满足时** | “fullId”等于“com.android.quicksearchbox:id/search_src_text”|
| **输入字段超时** | 3000 | 3000
| **输入内容** |更多登录反检测浏览器|
| **输入前清除字段** |上 |
| **输入后按回车** |关闭 |
| **查找元素后延迟** | 0 |

输出中的“输出参数”：

Element: { "isScrollable": false, "centerY": 35, "resourceId": "com.android.systemui:id/clock", "centerX": 71, [待补]
