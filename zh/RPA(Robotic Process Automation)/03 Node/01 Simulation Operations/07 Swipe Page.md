# 滑动页面

## 参数

- **[滑动开始坐标]**
    - X（数字）
    - Y（数字）

    您可以通过开发者工具获取云手机中的X、Y坐标值。

- **[滑动结束坐标]**
    - X（数字）
    - Y（数字）

    您可以通过开发者工具获取云手机中的X、Y坐标值。

- **[坐标随机偏移]**（数字）

    默认为0。设置偏移值后，每次起始位置和结束位置都会在偏移范围内随机上、下、左、右偏移。

    例如，设置X：50，Y：100，随机坐标偏移量为10，则X的范围为40px-60px，Y的范围为90px-110px。

- **[滑动持续时间]**
    - 从起点到终点滑动的持续时间（以毫秒为单位）。

- **[滑动后延迟]**

## 输出

- **输出参数**
    - 无
- **执行日志**
    - 开始时间
    - 结束时间
    - 持续时间（毫秒）
    - 状态（成功/失败）

## 示例

例如，在云手机桌面上，向上滑动至APP抽屉。通过开发者工具找到开始和结束位置（开始X：270，Y：1500；结束X：270，Y：1150）。参数如下：


- **滑动开始坐标**
    -X：“270”
    - 是：“1500”
- **滑动结束坐标**
    -X：“270”
    - 是：“1150”
- **坐标随机偏移**：“0”
- **滑动后延迟**：“0”
