# 运行亚行外壳

## 参数

- **[运行 ADB Shell]** 输入 ADB 的脚本命令，[什么是安卓 ADB 命令？](https://developer.android.com/tools/adb)
    
- **[超时]**：设置命令的最长运行时间。当超过最长时间时，将跳过当前步骤。

## 输出

- **输出参数**
    - 无
- **执行日志**
    - 开始时间
    - 结束时间
    - 持续时间（毫秒）
    - 状态（成功/失败）
