# 下载文件到手机

## 参数

- **[文件网址]**
    - 您的文件的地址。当前流程会根据这个地址将文件下载到对应的手机目录中。
- **[下载到]**
    - 专辑：`/sdcard/DCIM/Camera/`
    - 下载：`/sdcard/Download/`
    - 指定目录

## 输出

- **输出参数**
    - 无
- **执行日志**
    - 开始时间
    - 结束时间
    - 持续时间（毫秒）
    - 状态（成功/失败）
