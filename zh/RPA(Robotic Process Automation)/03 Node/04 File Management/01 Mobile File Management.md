# 移动文件管理

## 参数

- **操作模式**
    - 创建新文件夹
    - 清除文件夹
    - 删除特定文件
- **指定目录**


---

#### Android常用路径供参考：

| 文件夹 | 路径 |
| ------------------ | ---------------------- |
|专辑路径| `/sdcard/DCIM/相机/` |
|下载目录 | `/sdcard/下载/` |

## 输出

- **输出参数**
    - 无
- **执行日志**
    - 开始时间
    - 结束时间
    - 持续时间（毫秒）
    - 状态（成功/失败）
