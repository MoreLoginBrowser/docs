# 切换

## 参数

您可以通过不同的条件创建多个分支。在每个分支中输入相应的条件；当满足条件时，流程将进入当前分支。

- **当条件满足时**
    - 可以输入文本或添加变量，支持多种逻辑判断条件：
        - 等于
        - 不等于
        - 大于
        - 小于
        - 大于或等于
        - 小于或等于
        - 包含
        - 不包含
        - 存在
        - 不存在（逻辑运算右侧内容无需填写）

## 输出

- **输出参数**
    - 无
- **执行日志**
    - 开始时间
    - 结束时间
    - 持续时间（毫秒）
    - 状态（成功/失败）
