# HTTP 请求

## 参数

- **[请求方式]**
    - 获取
    - 后
    - 放置
    - 删除
- **[请求网址]**
    - 输入您要使用的地址。
- **[请求标头]**
    - 使用此参数在请求中发送标头。标头包含有关请求的元数据或上下文信息。
- **[查询参数]**
    - 查询参数充当 HTTP 请求的过滤器。如果您使用的API支持查询参数并且您的请求需要过滤器，请启用此选项。
- **[身体输入类型]**
    - 如果您需要在API请求中发送请求正文，请启用此选项。
- **[身份验证类型]**
    - 无授权
    - API 密钥
    - 基本身份验证


- **[超时]**
    - 当前节点运行的最长时间。

<br>

> [!提示]
> 对于 HTTP 请求，请参阅您的服务的 API 文档以获取详细指导。


## 输出

- **输出参数**
    - 无
- **执行日志**
    - 开始时间
    - 结束时间
    - 持续时间（毫秒）
    - 状态（成功/失败）
